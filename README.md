# ABC Live Web

ABC Live is a website that fetches latest BFSI news from Economic Times BFSI and Business Line, along with current rates from RBI, and presents them.

For the API fetching the news headlines from these sources, refer to [ABC Live Web API](https://github.com/Areen-Rath/ABC-Live-Web-API).

To go to the website, click  [here](https://abclive-bfsi.vercel.app/).

# Running Locally

The website is built on React using Next.js framework.PNPM is used as the package manager.

```bash
# Clone the project
git clone https://github.com/Areen-Rath/ABC-Live-Web.git

# Change directory
cd ABC-Live-Web

# Install dependencies
pnpm install

# Start development environment
pnpm dev
```

The project will then run locally on your device.
